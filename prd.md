## Intelligent Analyzer for Moroccan Public Tenders

Version: 1.0  
Project Type: Final Academic Project (Fil Rouge)  
Author: Student Project – AI Developer Training  
Date: 2026

---

# 1. Project Overview

TenderScope AI is a simplified intelligent platform designed to help Moroccan SMEs quickly evaluate whether a public tender (appel d'offres) is relevant to their company profile.

The system analyzes tenders and compares them with a company profile using Natural Language Processing (NLP) and semantic similarity.

The goal is to reduce the time SMEs spend analyzing public tenders manually.

This project is implemented as a **minimal viable product (MVP)** to demonstrate AI and full-stack development competencies required for the training program.

---

# 2. Problem Statement

In Morocco, many public tenders are published by government institutions.

SMEs face several problems:

- Large number of tenders to review
- Manual analysis is time-consuming
- Many tenders are irrelevant to their business sector
- Lack of intelligent tools to filter and prioritize tenders

Therefore, companies waste time analyzing tenders that do not match their capabilities.

---

# 3. Project Objective

The objective of TenderScope AI is to create a simple intelligent tool that:

1. Stores public tender information
2. Stores a company profile
3. Uses AI to measure semantic similarity
4. Calculates a compatibility score between the company and each tender
5. Displays a ranked list of tenders based on relevance

The system does **not attempt to fully automate tender analysis**, but provides **decision support**.

---

## 6.3 AI Matching Engine

The system compares:

Company profile description  
with  
Tender description

Using semantic embeddings generated by a language model.

The similarity between the two texts is computed using cosine similarity.

The output is a compatibility score between:

0 and 100

Example:

Tender A → 82% compatibility  
Tender B → 14% compatibility

---

# 11. AI Matching Method

The AI engine uses semantic embeddings.

Steps:

1. Convert company profile text into vector embedding
2. Convert tender description into vector embedding
3. Compute cosine similarity between vectors
4. Convert similarity into percentage score

Formula:

Score = cosine_similarity * 100
